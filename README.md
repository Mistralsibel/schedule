# 曦序日程
简约文艺风静态前端日程管理网页，数据本地存储，开箱即用。

## ✨ 功能清单
### 基础功能
- 日程新建、编辑、标记完成、删除
- 任务分类筛选，卡片式日程展示
- 数据导出、一键清理已完成任务
- 浏览器本地持久化存储

### 创意特色
- 昼夜双主题切换：晨光日间 / 星空夜景两套场景
- 自定义页面名称，支持15字限制文艺励志铭
- Three.js 3D许愿灵树，完成任务生成发光果实
- 沉浸式专注模式
- 莫兰迪磨砂玻璃UI，细腻微交互动效

## 🚀 运行部署
1. **本地使用**：浏览器直接打开 `index.html`
2. **公网分享**：支持部署 Vercel / 阿里云OSS / Gitee Pages 等静态平台

> ⚠️ 部署提示：使用OSS图片资源需要配置CORS跨域策略

## 🛠 技术栈
HTML + CSS + 原生JavaScript + Three.js
纯前端项目，无需后端服务

## 📌 使用须知
- 清理浏览器缓存会丢失日程，更换设备前请导出数据备份
- 浏览器无痕模式关闭后不会保留任何数据
- 仅限个人非商用使用
